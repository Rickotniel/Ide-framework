# Ide-framework 
# Projet : TaskFlow
 une plateforme logicielle destinée à la gestion et au suivi des livrables au sein d’une équipe technique.
L’application doit permettre d’ajouter des tâches, de modifier leur état d’avancement, de
persister les données localement et de naviguer sur des fiches descriptives détaillées sans
aucun rechargement de page.

Guide simple d'installation et de démarrage.

## Prérequis

- Node.js (recommandé ≥ 18)
- npm (fourni avec Node.js)

## Installation

Ouvrez un terminal à la racine du projet (le dossier contenant `package.json`) et exécutez :

```bash
npm install
```

Cette commande crée le dossier `node_modules` et installe toutes les dépendances.

## Lancer le serveur de développement

```bash
npm run dev
```

Par défaut l'application est disponible sur :

http://localhost:5173

## Build de production (optionnel)

```bash
npm run build
npm run preview
```
